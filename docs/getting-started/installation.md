# Installation

## From PyPI

```bash
pip install eto
```

## From Conda

```bash
conda install -c mullenkamp eto
```

## Requirements

- Python >= 3.8
- NumPy

## Development Install

Clone the repository and install with [uv](https://docs.astral.sh/uv/):

```bash
git clone https://github.com/mullenkamp/ETo.git
cd ETo
uv sync
```

Run tests:

```bash
uv run pytest
```
